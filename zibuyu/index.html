what is this?

I manually copy the 子不语 story from the book into a json file, which I input into a python script that adds the generated pinyin and English translation and outputs to a different json file. Such output json is used to generate this page, which I use as aid for reading the book.

The google translation isn't the best... but it's enough to clarify doubts or learn new words.

libraries used:

  • jieba → segments chinese texts into words
  • pypinyin → generates pinyin
  • pypinyin-dict → improves pinyin generation
  • deep-translator → translates using google
